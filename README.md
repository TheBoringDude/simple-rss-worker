# simple-rss-worker

Just a Simple RSS Parser API built with cloudflare worker

## Usage

```
curl -X POST https://rss.theboringdude.workers.dev/ -H "Content-Type: application/json" -d '{"url": "https://data.gmanetwork.com/gno/rss/news/feed.xml"}'
```
